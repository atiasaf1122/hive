# Icons

The icon set is generated by `desktop/scripts/generate_icons.py` and
**committed to git** so `tauri:dev` and `tauri:build` work on a fresh
checkout without anyone needing Python.

Contents:

| File              | Size      | Used by                    |
|-------------------|-----------|----------------------------|
| `icon.png`        | 1024×1024 | Tauri runtime + masters    |
| `32x32.png`       | 32×32     | Linux                      |
| `128x128.png`     | 128×128   | Linux                      |
| `128x128@2x.png`  | 256×256   | Linux (HiDPI), macOS       |
| `icon.ico`        | 16/32/48/64/128/256 | Windows resource + .msi |
| `icon.icns`       | multi     | macOS .dmg                 |

To regenerate after changing the logo:

```bash
python desktop/scripts/generate_icons.py
```

(Requires Pillow.)
